# Sistema de Gestión Básica de Productos

Ejercicio integrador desarrollado para el curso de Python en **Talento Tech**.

## Descripción

Programa de consola que permite gestionar una lista de productos. El usuario puede agregar, visualizar, buscar y eliminar productos a través de un menú interactivo.

## Funcionalidades

- **Agregar producto** — ingresa nombre, categoría y precio con validación de datos
- **Mostrar productos** — lista todos los productos registrados con numeración
- **Buscar producto** — búsqueda por nombre, sin distinción de mayúsculas/minúsculas
- **Eliminar producto** — elimina un producto por su número en la lista
- **Salir** — cierra la aplicación

## Tecnologías

- Python 3

## Cómo ejecutar

```bash
python gestion_productos.py
```

## Conceptos aplicados

- Listas y sublistas
- Bucles `while` y `for`
- Condicionales
- Validación de entradas del usuario
- f-strings
- `enumerate()` para listados numerados
